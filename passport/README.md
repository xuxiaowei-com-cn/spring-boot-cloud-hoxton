# Passport（服务模块）

## dependencies

- Developer Tools
    - Lombok
    - Spring Configuration Processor

- Web
    - Spring Web
        - 提供 Web 服务
    - Spring Session

- Template Engines
    - Thymeleaf

- SQL
    - JDBC API
    - MyBatis Framework
    - MS SQL Server Driver
    - MySQL Driver
    - Oracle Driver

- NoSQL
    - Spring Data Redis(Access+Driver)

- Spring Cloud Security
    - Cloud Security

- Spring Cloud Config
    - Config Client
        - 提供读取配置服务器中的配置

- Spring Cloud Discovery
    - Eureka Server
        - 提供 @EnableEurekaServer 注解
        - 包含 Ribbon 依赖
